SQL Server でクエリをジョブ化する

SQLサーバで何十分もかかるストアドプロシージャを実行すると、実行クライアントを開けっぱなしにしないといけないのが不便になることがある。 SQL Server Agent を使うと、裏でジョブ実行できて便利。

ジョブ化したいSPは以下： 


CREATE DATABASE testdb;
GO

USE testdb;
GO

CREATE TABLE dbo.test_table (
    id INT NOT NULL IDENTITY PRIMARY KEY
    , field SYSNAME
    , created_at DATETIME NOT NULL DEFAULT GETDATE()
);
GO

/* このSPをジョブ化したい */
CREATE OR ALTER PROCEDURE dbo.usp_test_table_insert
    @field SYSNAME
AS
    INSERT INTO dbo.test_table (field) VALUES (@field);

実行ジョブ作成： 


USE msdb;
GO

EXEC dbo.sp_add_job
    @job_name = N'test_job';

EXEC sp_add_jobstep
    @job_name = N'test_job',
    @step_name = N'process step',
    @subsystem = N'TSQL',
    @command = N'test.dbo.usp_test_table_insert'; -- ここでジョブ化したいSP名指定

EXEC dbo.sp_add_jobserver
    @job_name =  N'test_job';

ジョブ実行： 


DECLARE @ReturnCode int;
EXEC @ReturnCode = dbo.sp_start_job N'test_job';
PRINT @ReturnCode; -- 0ならOK、1ならエラー
GO

SQL Server Agent を LinuxベースのDockerコンテナ 版SQL Serverで動かすためには、SQL Server Agentの機能をオンにしないといけない。SQLサーバ再起動が必要な作業なので、Dockerfileを新規作成します。 


FROM mcr.microsoft.com/mssql/server:2017-latest
RUN /opt/mssql/bin/mssql-conf set sqlagent.enabled true
